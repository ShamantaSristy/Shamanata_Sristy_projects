1.Elaboration of the word CSS

CSS is Cascading Style Sheet.Mainly used to add some color and other beautification to a website in order to look the website more attractive..

2.What is style tag?

Style tag is that tag which is used to style contents written in HTML.With the help of style tag we can add color in a html document or in other words in our website.

3.CSS colors.

In CSS,we use different types of color defined by hexcode color,rgb and rgba.Hexcode color indicades 3 shades of color,one of them is hexadesimal number.RGB color defines red,green,blue.RGBA defines red,green,blue and opacity or thickness of a color.

4.Background Color

We can add Background color of a text using CSS.

5.Height,Width,CSS measuring units

We can determine a text's font,font size,font color,height,width,and all these are measured in some parameters called pixel,in,cm,%,

6.ID,Class,Span tag

We can also determine color,different font,font size,color,background color of a text or some words in a line or only some selected texts using Id tag,span tag and class tag.

7.Border,Border radius,Margin

There's also option for adding border around texts,border radius if we want more beautification for our texts.And margin tag is the aother very important element to take that beautification to a next level.

8.Padding,Box style,Text alignment

Padding mean adding margin between the text and border.And together all attributes like margin,padding,height,width,border makes CSS Box Style.

9.External CSS

We use this to add a different CSS style sheet in our html file.This file adding process uses the link:css tag.
